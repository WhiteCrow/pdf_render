= PdfRender

This project uses MIT-LICENSE.

##Getting started

```ruby
gem 'pdf_render'
```

After you use bundle install pdf_render, you can call use pdf_reder like it:

```ruby
  def index
    respond_to do |format|
      format.html
      format.pdf {render pdf: "contents", template: "home/index"}
    end
  end
```

##Run test
run
```ruby
rake test
```
in gem's dirctory
